# 📊 Insurance Claims Analysis Dashboard

## 🔍 Project Overview

This project analyzes insurance data to identify claim patterns, customer behavior, and risk factors using Power BI.

## 📋 Business Requirements

* Analyze insurance policies and claims
* Track claim frequency and total claim amount
* Identify high-risk customers
* Understand impact of demographics and car usage

## 📌 Key KPIs

* Total Policies
* Total Claim Amount
* Claim Frequency
* Average Claim Amount

## 📈 Dashboard Insights

* Claim analysis by car usage and car make
* Customer segmentation by age and gender
* Trend analysis over time
* Risk analysis based on driver profile

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
