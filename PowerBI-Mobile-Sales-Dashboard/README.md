# 📊 Mobile Sales Dashboard (Power BI)

## 🚀 Overview
This project is a simulated real-world mobile sales dashboard built using Power BI.

## 🔹 Features
- DAX: MTD, QTD, YTD, SAMEPERIODLASTYEAR
- KPIs: Total Sales, Quantity, Transactions
- Drill-through navigation
- Power BI Q&A
- Interactive slicers and visuals

## 🛠 Tools
Power BI | DAX | Power Query | Excel

## 🤖 Approach
Used Gen AI tools (ChatGPT) for DAX formulas and dashboard design
